# Pixel Array Image Generator

[中文说明](#中文说明) | [English](#english)

## 中文说明

一个轻量的单文件 HTML 工具，用于绘制小型像素阵列图案，并将其按平铺方式导出为 PNG 或 BMP 图片。

### 功能

- 通过可点击网格绘制基础像素图案
- 支持 10 组 RGB 颜色预设
- 可自定义输出宽高
- 支持导出 PNG 和 BMP
- 在页面内实时预览生成结果

### 使用方法

1. 直接用浏览器打开 `pixel_array.html`
2. 设置网格行列数
3. 选择颜色并在网格中绘制
4. 设置输出尺寸与格式
5. 点击“生成图片”并下载结果

### 文件

- `pixel_array.html`: 主程序，包含界面、逻辑与样式

## English

A lightweight single-file HTML tool for drawing small pixel-array patterns and exporting them as tiled PNG or BMP images.

### Features

- Draw a base pixel pattern on a clickable grid
- 10 editable RGB color presets
- Custom output width and height
- Export to PNG and BMP
- Live in-page preview before download

### Usage

1. Open `pixel_array.html` in a browser
2. Set the grid rows and columns
3. Pick colors and paint on the grid
4. Configure output size and format
5. Click `Generate Image` and download the result

### Files

- `pixel_array.html`: main app containing UI, logic, and styles
